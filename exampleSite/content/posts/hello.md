Welcome to the Hugo Theme Deck! This theme is designed to be clean, modern, and easy to use.

## Features

- Multiple content types (Write, Quote, Now, Read, TIL)
- Dark mode support
- Responsive design
- RSS feed
- SEO optimized
- Image lazy loading
- Tag support

## Getting Started

1. Install Hugo
2. Create a new site
3. Add this theme
4. Configure your site
5. Start writing!
